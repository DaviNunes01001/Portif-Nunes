## Sobre o projeto

Versão 2.0 do meu portfólio pessoal, desenvolvida com **React + Vite**.

A principal evolução em relação à versão anterior foi a adoção de **componentização**, dividindo a interface em partes reutilizáveis e independentes — tornando o código mais organizado, fácil de manter e preparado para crescer.

## Como executar localmente

```bash
# Clonar o repositório
git clone https://github.com/davinunes01001/Portif-Nunes.git

# Entrar na pasta do projeto
cd Portif-Nunes

# Instalar dependências
npm install

# Rodar em desenvolvimento
npm run dev
```

Disponível em `http://localhost:5173`

## O que aprendi

Neste projeto pratiquei a construção de interfaces com React focando em componentização real — não só dividir arquivos, mas pensar na responsabilidade de cada componente. Também aprendi a fazer deploy contínuo com GitHub Pages integrado ao Vite.
